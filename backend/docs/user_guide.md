# User Guide (Operations)

## Month Workflow
1. Select `Club` and `Period`.
2. Post ledger transactions in Draft/Review:
   - contributions
   - withdrawals
   - income
   - expenses
   - adjustments (only in open periods)
3. Review investor balances and reconciliation stamp.
4. Submit for review.
5. Run close only when checklist and reconciliation pass.

## Lock Rules
- Closed periods are read-only.
- Any correction must be posted into a later period as adjustment.

## Reports
- Reports are generated from closed period context.
- Download immutable PDF snapshots from Reports screen.

## Intelligent Mode
- Analysis includes insights, anomaly flags, and scenario simulator.
- Copilot is read-only and scoped to selected club+period.
